## Информация

### Как запустить проект?

##### Для запуска проекта нужно прописать следующие команды:
`Linux`
```bash
python -m venv .venv
source .venv/bin/activate
pip install aiogram
python run.py
```
###### На некоторых системах нужно вместо python писать python3

`Windows`
```bash
python -m venv .venv
.venv\Scripts\activate
pip install aiogram
python run.py
```
